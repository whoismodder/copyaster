# Changelog

## [0.1.0] - 2026-04-15

### Agregado
- Menu bar app con ícono clipboard + "C"
- Historial de últimos 50 clips (auto-eliminación del más viejo)
- Guardados persistentes con título y emoji
- Multi-select con ⌘+click para copiar varios clips juntos
- Hover preview sobre el ícono
- Selector inline con ⌘⇧V (configurable)
- Navegación con teclado (↑↓ Enter Escape Tab)
- Búsqueda en tiempo real
- Exportar a Apple Notes
- Universal Clipboard — detecta clips del iPhone
- Detección de passwords (ConcealedType/TransientType)
- Arranque automático con macOS
- Ajustes: hotkey configurable, login at launch
- Click derecho → Ajustes / Salir
